Pareto-based Evaluation of Lightweight AI Models on Embedded Devices

Authors
Patricio Rojas Carrasco

Description
This repository contains the reproducibility package for the study 
on multi-objective evaluation of lightweight AI models deployed on 
embedded devices.

Models evaluated
- Multinomial Logistic Regression (MLR)
- Multilayer Perceptron (MLP)
- Reduced Convolutional Neural Network (CNN)

Platforms
- ESP32-S3
- Raspberry Pi Pico
- Raspberry Pi Zero

Metrics
- Accuracy
- Latency
- Memory footprint
